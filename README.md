DVLD - Driving License Management System

Project Description

DVLD is a comprehensive system designed to streamline the process of issuing, renewing, and managing driving licenses for users. It allows employees to perform various operations such as issuing new licenses, renewing existing ones, suspending and reinstating licenses, and issuing replacements for lost or damaged licenses. The system supports both local and international licenses.

Key Features

- Issue a new driving license (local and international).
- Easily renew an existing driving license.
- Suspend and reinstate licenses as needed.
- Issue replacement licenses for lost or damaged ones.
- Comprehensive user management system with different roles.
- Driver and user account management.
- Secure login system with active/inactive status verification.

Technologies Used

- Programming Language: C# (.NET)
- Database: SQL Server
- Data Access: ADO.NET
- User Interface: Windows Forms (WinForms)

Project Structure:

- Data Access Layer (DAL): Handles database operations.
- Business Logic Layer (BLL): Implements business logic.
- Presentation Layer (WinForms UI): Displays data and interacts with users.

How to Run the Project

1. Download the project from GitHub.
2. Open the project in Visual Studio.
3. Update the database connection settings in the configuration file.
4. Run the database using SQL Server.
5. Execute the application and explore its features.
